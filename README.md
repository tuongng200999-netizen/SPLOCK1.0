### Cài Đặt & Build

Xem chi tiết tại: [BUILD_GUIDE.md](BUILD_GUIDE.md)

```bash
# Cài đặt
npm install

# Build web
npm run build

# Thêm Android
npm run cap:add:android

# Sync
npm run cap:sync

# Mở Android Studio
npm run cap:open:android

# Build APK
npm run cap:build:android
```
